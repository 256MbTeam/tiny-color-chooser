Tiny Color Chooser Usage

Color chooser with default settings 
new TinyColorChooser("color_chooser_1");
You can setup your own scope of colors 
new TinyColorChooser("color_chooser_2", {
                    colors: ["AAA","BBB","CCC","DDD","FFF"],
                    default_color: "BBB",
                    popup_height: 26
                });

Avaliable options for Tiny Color Chooser

Option Description
colors Array of avaliable colors
default_color Color, that be setted by defaults
popup_width
popup_height
element_classname
popup_classname
preview_classname
